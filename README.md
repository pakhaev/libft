# libft
Учебный проект.
Реализация стандартных функций языка C.

```
ft_atoi - приведение строки в числовой вид
ft_bzero - устанавливает первые n байтов области, начинающейся с s в нули (пустые байты)
ft_isalnum - проверяет, является ли аргумент c типа int буквой или цифрой
ft_isalpha - проверяет, является ли аргумент строчной или прописной буквой алфавита
ft_isascii - возвращает ненулевое значение, если c является символом ASCII (в диапазоне 0x00 — 0x7F)
ft_isprint - проверяет значение аргумента, передаваемого через параметр сharacter , представляет ли оно собой печатный символ
ft_memccpy - копирует содержимое одной области памяти в другую
ft_memchr - поиск первого вхождения указанного символа в массиве
ft_memcmp - сравнивает первые count символов массивов, адресуемых параметрами buf1 и buf2
ft_memcpy - копирует содержимое одной области памяти в другую
ft_memset - заполнение массива указанными символами
ft_strcat - добавляет в строку, на которую указывает аргумент destination, строку, на которую указывает аргумент append
ft_strchr - поиск первого вхождения символа в строку
ft_strcmp - сравнение строк
ft_strcpy - копирование нуль-терминированной строки (включая нуль-терминатор) в заданный буфер
ft_strdup - дублирование строк с выделением памяти под новую строку
ft_strlen - возвращает длину строки, переданной в качестве параметра
ft_strncat - объединение строк c ограничением длины добавляемой строки
ft_tolower - преобразование прописных букв в строчные
ft_toupper -  преобразование строчных букв в прописные
```
